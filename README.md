# Elite Shoes - Frontend

## Descripción
Aplicación web moderna para una tienda de calzado deportivo desarrollada con **React**. Permite a los usuarios navegar por categorías, agregar productos al carrito y realizar pedidos. Incluye panel de administración para gestionar el inventario.

---

## Tecnologías Utilizadas

Tecnología
**React**| Framework principal de UI |
**React Router**| Navegación entre páginas |
**Axios**| Conexión con API REST |
**CSS3**| Estilos y diseño responsive |
**Git/GitHub**| Control de versiones |
---

## Funcionalidades

### 👤 Usuarios (Clientes)
- Registro de nuevos usuarios
- Inicio de sesión seguro
- Navegación por categorías (Hombre, Dama, Infantil)
- Visualización de productos con imágenes
- Carrito de compras funcional
- Realización de pedidos
- Cálculo automático de totales

### Administradores
- Panel de administración exclusivo
- Creación de nuevos productos
- Eliminación de productos
- Visualización de inventario
- Gestión de stock

---

## Instalación

### Requisitos Previos
- Node.js (v14 o superior)
- npm o yarn
- Backend corriendo en http://localhost:8080

### Pasos de Instalación

```bash
# 1. Clonar el repositorio
git clone https://github.com/Jefersson2015/elite-frontend.git

# 2. Navegar a la carpeta del proyecto
cd elite-frontend

# 3. Instalar dependencias
npm install

# 4. Iniciar el servidor de desarrollo
npm start
